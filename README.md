# Life Reset Tracker | 人生重置打卡

基于 Dan Koe 的《[How to Fix Your Entire Life in 1 Day](https://www.youtube.com/watch?v=S8lR0TyMpkk)》协议，开发的习惯追踪打卡应用。

## 🌟 核心功能

- **GitHub 风格打卡日历** - 直观的绿点热力图，记录每一天的进步
- **连续打卡奖励** - 在 3/7/15/30/60/90/180/360 天自动触发里程碑奖励
- **每日杠杆任务** - 专注于每天最重要的 3-5 件事
- **数据本地存储** - 无需后端，所有数据保存在浏览器

## 🚀 快速开始

### 本地运行

1. 下载或克隆本项目
2. 直接在浏览器打开 `index.html`，或者使用简单的 HTTP 服务器：

```bash
# Python
python3 -m http.server

# Node.js
npx http-server
```

### 部署到 GitHub Pages

1. Fork 本仓库
2. 进入 Settings -> Pages
3. 在 "Source" 部分选择 `main` 分支作为源
4. 保存后稍等片刻，即可通过 `https://你的用户名.github.io/仓库名/` 访问

## 💫 每日使用协议

1. **早晨**：回顾反愿景和你的理想身份
2. **白天**：时刻保持觉知，完成每日杠杆任务
3. **晚上**：点击打卡，检查是否达成里程碑
